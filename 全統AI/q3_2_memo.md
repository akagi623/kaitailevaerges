# 問三⑵ バリエーション展開：回答用データ（平文・980文字版）

### AIとの対話の要点（約980文字）

自分: ⑴で生成したメインビジュアルを元に、Image-to-Image（i2i）で3つのバリエーションを展開したいです。ゲームキャンペーンとして、どのような差分を作るのが効果的でしょうか？
→ AI: ユーザーに「キャラクターの多面的な魅力」を伝えるため、以下の3つのバリエーションを提案します。1枚目は「表情差分（キリッとした戦闘顔）」、2枚目は「時間帯差分（夕暮れの空中都市）」、3枚目は「ポーズ差分（放影の瞬間）」です。これにより、静的な立ち絵から物語性が生まれます。
→ 自分: i2iを行う際、キャラクターのアイデンティティ（角、翼、空色×金の配色）を崩さないためのテクニックはありますか？
→ AI: Denoising Strength（描き込み強度）の調整が鍵です。設定を0.4〜0.6程度に抑え、元のシルエットを維持しつつ変化を加えます。さらに、ControlNetの「Canny」や「Lineart」を併用して細部の線画を固定し、プロンプトで「同じ衣装、同じ角のデザイン」を強調することで、別キャラ化を防ぎます。
→ 自分: 具体的な演出意図を教えてください。
→ AI: 表情差分では、瞳に光のハイライトを強めに入れ、戦う覚悟を感じさせます。時間帯差分では、背景の空色をオレンジと紫のグラデーションに変え、金の装飾に夕陽の照り返しを乗せることでエモーショナルな演出にします。ポーズ差分では、光の弓を大きく引き絞り、周囲に青い光の粒子が舞うエフェクトを追加して、キャンペーンの「動的な目玉」にします。
→ 自分: なるほど。元のクオリティを維持しながら、世界観を深める3枚の差分として構成するのですね。
→ AI: はい。単なる色替えではなく、各画像に異なる「感情」と「シチュエーション」を付与することで、一貫性のあるキャンペーン素材となります。これらの差分を並べることで、ユーザーにソラというキャラクターの成長や世界観への没入感を強く印象付けることができます。
→ 自分: よく理解できました。キャラクターの一貫性を保ちつつ、劇的な変化を生むプロンプトと設定案としてまとめます。

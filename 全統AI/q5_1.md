# FreshBox LP 制作要件定義とプロンプト設計 (q5_1.md)

## 1. 要件定義

### ターゲット層
- 30〜40代の共働き世帯
- 料理初心者、または献立作成や買い物に負担を感じている層

### LPの目的
- 「初回おためしセット（980円）」のコンバージョン（注文促進）

### セクション構成
1. **メインビジュアル (Hero)**: 
   - キャッチコピー：「献立の悩み、卒業。共働き夫婦に選ばれる、時短と美味しさの新体験。」
   - サブコピー：おためしセット980円（送料無料）
2. **共感セクション (Pain Point)**: 
   - 仕事終わりの買い物、献立作り、栄養バランスへの不安。
3. **解決策 (Solution)**: 
   - 旬の食材とレシピがセットで届く。買い物不要。
4. **選ばれる3つの理由 (Features)**:
   - ① 20分で完成する「時短レシピ」
   - ② プロが選んだ「旬の厳選食材」
   - ③ 心理的負担を取り除く「献立済み」
5. **お客様の声 (Social Proof)**: 
   - 利用者の満足度と具体的な口コミ。
6. **オファー/CTA (Closing)**: 
   - おためしセット内容のビジュアル表示と注文ボタン。

---

## 2. AIコーディングエージェントへの指示プロンプト

```markdown
あなたは、コンバージョン率を最大化するLP制作を得意とするシニアフロントエンドエンジニアです。
以下の要件に基づき、食材宅配サービス「FreshBox」のLPを制作してください。

### 技術スタック
- HTML5 / CSS3 (モダンなレイアウト)
- レスポンシブ対応 (モバイルファースト)
- Google Fonts (日本語：Noto Sans JP)

### デザインガイドライン
- 配色：食欲をそそる暖色系（オレンジ系）と清潔感のある白を基調にする。
- 雰囲気：明るい、親しみやすい、高品質。

### セクション構成
1. Header: ロゴと注文ボタン(CTA)
2. Hero: 魅力的な食材背景とキャッチコピー、980円のオファー。
3. Concept: 「共働きの夜を変える」というメッセージ。
4. Features: アイコン等を用いた3つの強み紹介。
5. Pricing: おためしセットの詳細(画像含む)と強力なCTAボタン。
6. Footer: 規約、特商法、コピーライト。

### 特記事項
- CTAボタンは、背景色とのコントラストを高くし、クリックしたくなるデザインにすること。
- 画像が不足している箇所は、プレースホルダー、または内容が伝わるフリー画像素材を仮置きすること。
- コードはクリーンで、保守性の高いクラス命名(BEM等)を採用すること。
```
